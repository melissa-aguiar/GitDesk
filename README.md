# GitDesk

 Projetos visando a reconstrução online de energia em aceleradores de partículas.
 
 ssf -> Projeto de uma arquitetura multicore de processadores em FPGA para operar um método de reconstrução de energia baseado em representação esparsa de dados em aritmética de ponto fixo.
 
 rede -> Projeto  de uma arquitetura multicore de processadores em FPGA para operar um método de reconstrução de energia baseado em redes neurais artificiais com tabela em aritmética de ponto fixo.

 rede_float -> Projeto  de uma arquitetura multicore de processadores em FPGA para operar um método de reconstrução de energia baseado em redes neurais artificiais com tabela em aritmética de ponto flutuante.

 rede_taylor -> Projeto  de uma arquitetura multicore de processadores em FPGA para operar um método de reconstrução de energia baseado em redes neurais artificiais com séries de Taylor em aritmética de ponto flutuante.
